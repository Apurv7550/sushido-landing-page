# Sushido Landing Page

A modern and responsive sushi restaurant landing page built using HTML, CSS, JavaScript, Vite, and AOS animations.

This project focuses on clean UI design, smooth layout structure, reusable CSS sections, and a visually appealing restaurant website experience.

Tech Stack
HTML5
CSS3
JavaScript
Vite
AOS Animation Library
