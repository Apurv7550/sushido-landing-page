# Sushido Landing Page

A modern sushi restaurant landing page built with HTML, CSS, JavaScript, Vite, and AOS animations.

This project was created to practice frontend layout, styling, animations, and clean project structure. It includes a hero section, navigation bar, food sections, testimonials, and a responsive landing page design.



## Features

- Modern restaurant landing page design
- Clean and responsive layout
- Hero section with image and text content
- Navigation bar
- Popular food section
- Trending sushi and drinks section
- Customer review section
- Smooth scroll animations using AOS
- Organized CSS files by section
- Fast development setup with Vite

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vite
- AOS Animation Library

## Getting Started

To run this project locally, follow these steps.

### 1. Clone the repository

```bash
git clone https://github.com/Apurv7550/sushido-landing-page.git
```

### 2. Open the project folder

```bash
cd sushido-landing-page
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

Open the local server link shown in the terminal.
